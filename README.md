# Global Sales & Manufacturing Analysis Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze global sales performance, manufacturing costs, profit trends, and product-wise performance.

## Tools Used
- Power BI
- DAX
- Data Modeling
- Excel

## Key Insights
- Monthly sales and profit trends
- Manufacturing cost analysis
- Country-wise performance comparison
- Product-wise sales distribution
- Gross sales and discount analysis
